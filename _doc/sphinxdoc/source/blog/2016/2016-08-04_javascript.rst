    La présentation
    `10 plotting libraries <http://www.xavierdupre.fr/app/jupytalk/helpsphinx/2016/pydata2016.html>`_
    montre les principales librairies de dessin interactif
    mais cela toujours une aventure de mélanger Javascript et Python.
    Via le site d'`Alan Richmond <https://python.zeef.com/alan.richmond>`_,
    j'ai découvert cette page
    `PyScript vs X <https://github.com/zoofIO/flexx/wiki/PyScript-vs-X>`_
    qui listent différentes stratégies pour convertir, utiliser,
    interagir. Il faut partie de la librairie
    `flexx <https://flexx.readthedocs.io/en/stable/>`_
    dont les `exemples <https://flexx.readthedocs.io/en/stable/ui/examples.html>`_
    sont plutôt bluffants. Il est même possible d'interagir avec
    `bokeh <https://flexx.readthedocs.io/en/stable/ui/examples/with_bokeh_src.html#with-bokeh-py>`_,
    et on peut aussi réaliser des graphiques
    `interactifs <https://flexx.readthedocs.io/en/stable/ui/examples/twente_src.html#twente-py>`_.
