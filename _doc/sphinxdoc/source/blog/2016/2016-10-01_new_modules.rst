

.. blogpost::
    :title: Découvrir de nouveaux modules
    :keywords: distribution, package
    :date: 2016-10-01
    :categories: module
    
    La planète Python évolue très vite et il est quasiment impossible
    d'essayer tous les modules qu'on découvre voire même de les découvrir tout court.
    Une source assez fiable consiste à regarder la liste des packages présents
    dans les distributions `WinPython <https://winpython.github.io/>`_ ou 
    `Anaconda <https://docs.continuum.io/anaconda/pkg-docs>`_. De plus, 
    lorsqu'un module est ajouté, cela signifie qu'il est assez abouti
    pour être distribué.
