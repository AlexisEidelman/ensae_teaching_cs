\partietdc{régression linéaire}

La régression linéaire est une façon de relier une variable $Y$ à une autre $X$. On écrit le modèle suivant~:

\begin{eqnarray}
Y = \alpha X + \beta + \epsilon
\end{eqnarray}
 
Pour estimer les coefficients $\alpha$ et $\beta$, on utilise plusieurs observations (ici 360) puis on minimise l'écart $\epsilon_i$ entre la valeur observée $Y_i$ et la valeur estimée $\hat{Y_i}=\alpha X_i + \beta$ pour chaque observation~$i$. Si les écarts sont indépendants et distribués selon une loi normale, cela revient à minimiser l'expression suivante~:

\begin{eqnarray}
E &=& \underset{\alpha,\beta}{\min} \pa{\sum_i \epsilon_i^2} \\
	&=& \underset{\alpha,\beta}{\min} \pa{\sum_i \pa{ Y_i - (\alpha X_i + \beta)}^2}
\end{eqnarray}

Afin de simplifier l'écriture du problème, on suggère de considérer une seconde variable constante $X_2=1$ et d'écrire~:

\begin{eqnarray}
\alpha X + \beta = \alpha_1 X_1 + \alpha_2 X_2 =  X \pa{\alpha_1,\alpha_2} = X A  
\end{eqnarray}

Où $X$ est la matrice composée des colonnes $(X_1,X_2)$. Dans ce cas, on résoud ce problème en annulant la dérivée et $A$ vaut~:

\begin{eqnarray}
A = \pa{X'X}^{-1} X' Y 
\end{eqnarray}

\tdquest Il est possible de "tester" la nullité d'un coefficient pour vérifier si une variable est importante lors de la régression\footnote{voir \textit{Probabilités, Analyse des Données et Statistiques} de Gilles Saporta, Editions Technip.}. En particulier pour la dénivellation, on cherche à savoir si le coefficient est presque nul ou pas. On calcule pour cela la valeur~:

\begin{eqnarray}
t_{n-p-1} &=& \frac{ a_j } 
								   { \sqrt{ \frac{ \norm{ y - XA }^2} { n-p-1 } (X'X)^{-1}_{jj} }}
\end{eqnarray}

Où $n$ est le nombre d'observations, $p$ la dimension du problème, $M_{jj}$ le $j^{\text{ieme}}$ coefficient sur la diagonale. Ce coefficient suit une loi de Student\footnote{http://fr.wikipedia.org/wiki/Loi\_de\_Student}. Il suffit de comparer la valeur obtenue avec la table de la loi de Student.
